Descripción: Proyecto de práctica donde analizo una base de datos de 10,000 registros y 2,000 productos para optimizar el control de almacén.

Habilidades aplicadas:

Limpieza de datos: Eliminación de duplicados y normalización de categorías.

Análisis Descriptivo: Identificación de productos con baja rotación.

Visualización: Creación de un resumen ejecutivo con los KPIs principales (Stock total, Valor de inventario y Top ventas).

Herramientas: Excel Avanzado / Tablas Dinámicas.
